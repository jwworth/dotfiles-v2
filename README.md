These are my dotfiles. They are heavily inspired by the [Hashrocket
Dotmatrix][dotmatrix].

### Setup (new machine)

- Install iTerm and turn on dark colors via profiles
- Install homebrew
- Install vim-plug
- Authenticate with Github and clone the repo

Then:

```
./bin/install
```
